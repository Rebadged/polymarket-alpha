# Research Brief: Geography / Location Strategy for an Ambient Channel Fleet

> For a deep-research instance. Self-contained. Operator "CK" is a **Canada-based
> solo creator** building a fleet of faceless AI ambient YouTube channels (lead
> channel = "Rainy Cabin Ambience": rain/storm + cozy-cabin loops, real cinematic
> motion via Higgsfield, original Suno audio, multi-hour seamless loops, YouTube-
> first monetization, Facebook Reels secondary; budget ~$100 CAD/mo). YouTube is
> the revenue engine; TikTok Creator Rewards excludes Canada.

## The idea to evaluate

CK proposes biasing channel content toward specific geographies to "game" RPM —
e.g. cozy cabins set in named cold-climate places (upper British Columbia, the
Rockies, Norwegian fjords, Scottish Highlands) — on the intuition that targeting
high-revenue regions (Canadian/EU/US) while building the channel could lift
earnings.

## The mechanic to verify FIRST (don't optimize the wrong thing)

Working assumption (confirm or correct with evidence): **YouTube RPM/CPM is
determined by the VIEWER's country, not the creator's country and not the
location depicted in the video.** If true, "Canadian content" does not by itself
raise RPM, and Canada's CPM (~$5.71 median) is below the US (~$10–12). The real
levers would then be (a) acquiring viewers from high-CPM English markets,
(b) seasonality, and (c) SEO/differentiation — NOT the creator's or the scene's
nationality. Please validate this framing before answering the rest.

## Questions

1. **RPM mechanic.** Is RPM purely viewer-geography? Any second-order effects —
   does content language, depicted location, or advertiser content-targeting shift
   the CPM for the same viewer? How much does audience-geo mix actually move RPM in
   the ambient/sleep niche specifically?
2. **Does location theming move the audience-geo mix?** Is there evidence that
   content set in / titled for a region (e.g. "Scottish Highlands rain") measurably
   attracts more viewers from that region or from high-CPM English markets, vs.
   generic "rain sounds"? Quantify if possible.
3. **Which markets to target.** Rank the realistic acquisition targets for ambient
   (US, UK, CA, AU, Western EU English speakers). Is chasing US viewers via theme/
   title/upload-time the highest-RPM play? Where does CA/EU fit?
4. **Seasonality.** Magnitude and timing of RPM seasonality (Q4–Q1) and of demand
   for cozy/winter/rain content by hemisphere. Should the fleet weight cozy content
   to Northern-Hemisphere winter, counter-program with summer content, or both?
   Is there a Southern-Hemisphere (AU) counter-season opportunity?
5. **Named-location SEO for ambient.** Do named real locations capture meaningful
   YouTube search volume vs generic terms ("rain sounds", "rain for sleeping")?
   What's the right TITLE convention — short place name ("British Columbia") vs
   descriptive phrase — and does adding a place help or hurt CTR/reach? (We found
   long location phrases make clunky titles; advise on phrasing.)
6. **Targeting vs reach tradeoff.** Does niche-location content shrink the
   addressable audience enough to reduce total watch-hours (and thus revenue),
   even if per-view RPM is similar? Where's the break-even?
7. **Canadian-creator specifics (non-RPM).** Beyond RPM: payment/tax (AdSense,
   W-8BEN/treaty), program eligibility, Facebook Content Monetization in Canada,
   any CRA/self-employment considerations that change the plan. (Known: TikTok
   Creator Rewards excludes Canada; YouTube YPP is available.)
8. **Concrete recommendation.** Should "Rainy Cabin Ambience" rotate a curated set
   of named Tier-1-relevant locations? If yes, give a specific curated location
   list + title format. If no, what's the better differentiation/SEO axis?

## Deliverable

A concrete geography/location strategy: (a) the corrected RPM mechanic in one
paragraph, (b) target audience-geography and how to acquire it, (c) whether/how to
use named locations (with a curated list + title convention if recommended),
(d) seasonal content weighting, (e) any Canadian-creator payment/tax/eligibility
notes. Tie back to: ambient niche, YouTube-first + Facebook-secondary, solo
operator, ~$100 CAD/mo, the per-video-variation (inauthentic-content) requirement.

## Fixed context (don't re-litigate)

Pipeline, tooling (Higgsfield motion + Suno + ffmpeg + YouTube API + VPS), the
3-channel slate (rain/cabin, sleep music, pet-dog), seamless-loop + −14 LUFS, and
the human review gate are all settled. This brief is ONLY about geography/location.
